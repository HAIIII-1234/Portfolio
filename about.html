About Me

Hi! I’m Afnan — a Grade 7 student diving deep into the creative world of web design. I love using HTML and CSS to turn simple ideas into colorful, working websites.

My journey started with basic tags and styles, but now I enjoy experimenting with layouts, animations, and responsive design. Every time I open my code editor, I learn something new!

I’m passionate about making websites that feel friendly and easy to use. My goal is to mix good design with clean, clear code so visitors enjoy every click.

Besides coding, I like sketching ideas in my notebook, learning from online tutorials, and challenging myself with small creative projects.
